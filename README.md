### 🚀 Sobre mim

Sou **Data Scientist I no Banco Bradesco**, trajetória construída de estagiário → analista de BI Jr → cientista de dados, ao longo de ~2 anos. Minha base foi construída em **modelos de saturação, análises ad hoc e clusterização para marketing**, e atualmente atuo no **departamento de consultoria interna** do banco.

- 🔭 Hoje aplico análise de dados e modelagem para apoiar decisões de negócio via consultoria interna
- 🧠 Background sólido em **saturação, ad hoc e clusterização** aplicados a marketing
- 🌱 Aprofundando estatística aplicada (distribuições de probabilidade, testes de normalidade, EBA + Bussab & Morettin) e me preparando para a **certificação Databricks**
- 💳 Interesse em expandir para **modelos de crédito, dados financeiros, dados governamentais e APIs públicas**
